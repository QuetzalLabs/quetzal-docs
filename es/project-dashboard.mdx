## Accediendo al Panel de Proyectos

Después de [registrarte](https://getquetzal.com/signup) en Quetzal, se te pedirá que crees un proyecto.

Una vez que tu proyecto esté creado, puedes encontrarlo en tu [Panel de Control de Quetzal](https://getquetzal.com/dashboard)

<Tip>
  Actualmente, Quetzal es gratuito para traducciones a otro idioma/localización.
  Si necesitas soporte para múltiples idiomas o un idioma que no está en la
  lista, por favor [contáctanos](https://getquetzal.com/contact)
</Tip>

## Uso del Panel de Proyectos

El panel de control es la forma principal de mantener un seguimiento de tus integraciones de Quetzal.

Allí puedes encontrar y gestionar el texto de tu aplicación, y ver todo el texto traducido.

### Cadenas del Proyecto

La [pestaña de cadenas](https://getquetzal.com/dashboard/strings/project/0) del proyecto lista todo el texto que se ha enviado a Quetzal para su traducción.

El texto de la cadena y sus traducciones pueden ser visualizados y editados desde esta página.

Cada cadena también muestra el archivo de tu código base donde fue encontrada (si es aplicable), junto con la fecha en que fue añadida.

### Lingo del Proyecto

La [pestaña de lingo](https://getquetzal.com/dashboard/lingo/project/0) del proyecto ofrece una forma de denotar palabras o frases especiales únicas de tu marca que deben ser traducidas de maneras particulares.

Puedes usar lingo para especificar traducciones personalizadas o evitar que se traduzca una frase en particular.

## Tareas Comunes

<AccordionGroup>
  <Accordion title="Encontrar tu Clave API">
    Tu clave API se puede encontrar en tu [página de configuración del
    proyecto](https://getquetzal.com/dashboard/settings/project/0). Tu clave API
    es importante y se utilizará para la mayoría de las integraciones, y otorga
    acceso para solicitar y recuperar texto de tu aplicación. La clave es única
    para tu proyecto y compartida entre todos los usuarios. Asegúrate de
    mantenerla privada.
  </Accordion>
  <Accordion title="Añadir Usuarios">
    La [pestaña de usuarios](https://getquetzal.com/dashboard/users/project/111)
    te permite invitar a usuarios para acceder y colaborar en tu proyecto de
    Quetzal.
  </Accordion>
  <Accordion title="Añadir Soporte para una Localización">
    Seleccionar 'Añadir Localización' en el lado izquierdo del panel de control
    te permite añadir soporte para un nuevo idioma a tu proyecto
  </Accordion>
</AccordionGroup>
